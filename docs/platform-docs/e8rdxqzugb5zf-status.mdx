## Status Definitions

- **Operational**: All systems are functioning normally
- **Degraded Performance**: Some services may be experiencing slower response times
- **Partial Outage**: Some services are unavailable
- **Major Outage**: Multiple services are unavailable
- **Maintenance**: Planned maintenance is in progress
