---
title: "Creating an app"
---

This guide will walk you through the process of creating your first app on our platform.

## Prerequisites

Before you begin, make sure you have:
- A developer account
- Basic understanding of programming
- Required development tools installed

## Step 1: Initialize Your Project

```bash
# Initialize a new app project
platform init my-app
cd my-app
```

## Step 2: Configure Your App

Edit the `app.config.json` file to set up your app's basic configuration:

```json
{
  "name": "My App",
  "version": "1.0.0",
  "description": "A sample app",
  "permissions": ["read", "write"]
}
```

## Step 3: Develop Your App

1. Create your main app file
2. Implement core functionality
3. Add necessary dependencies
4. Write tests

## Step 4: Test Your App

```bash
# Run tests
platform test

# Start development server
platform dev
```

## Step 5: Build Your App

```bash
# Build the app
platform build
```

## Next Steps

After creating your app, you can:
- Deploy it to the platform
- Add more features
- Integrate with other services
